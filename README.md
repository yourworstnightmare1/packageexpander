## Welcome to packageExpander

> [!NOTE]  
> If you are trying to install an app on Windows that uses an installer such as `.msi` or `.exe`, view what [appUnblocker](https://github.com/yourworstnightmare1/appunblocker) can do for you.

packageExpander allows you to install applications stored in `.pkg` files on macOS without administrator privliges. This allows the installation of apps like [FL Studio](https://www.image-line.com/) by Image Line which use package files.

> [!IMPORTANT]  
> packageExpander does not work on all package files. Tools such as [BlackHole](https://existential.audio/blackhole/) by [Existential Audio](https://www.existential.audio/) don't extract from the `.pkg` file's `Payload` file correctly.
